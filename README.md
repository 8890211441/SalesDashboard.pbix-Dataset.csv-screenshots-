# Sales Performance Dashboard (Power BI)

## Project Overview
This project is an interactive Sales Performance Dashboard built using Power BI. It helps analyze sales trends, revenue, profit, and customer performance through visual reports.

## Objectives
- Track overall sales performance
- Analyze profit and revenue trends
- Identify top-performing products
- Monitor customer and regional sales performance
- Support data-driven business decisions

## Tools Used
- Power BI
- Microsoft Excel / CSV
- Data Visualization
- Data Cleaning

## Key Insights
- Total Sales Analysis
- Profit Analysis
- Top Products Performance
- Regional Sales Distribution
- Customer Performance Tracking
